Parcel Radius Search

Walton's Parcel Radius Search was developed to allow user's the ability to select all parcel's of land within a specified radius. Once the user has completed the radius a javascript function is triggered to capture the points latitude, longitude and distance from the point(radius) in miles. On success, an ajax request to Geoserver is made and returns an array of geojson data. The parcel geometry stored in the PostgreSQL database is rendered in the Leaflet.js framework creating a selection of multiple polygons (parcels of land) within the specified radius.

Welldata Mapping Application

My Welldata Application system was developed to map out all recorded oil/gas well data throughout Alberta and Saskatchewan. The current system available to the public was out dated and extremely slow. The challenge was creating a fast and responsive application system for mobile, tablet and desktop that can handle over 100,000 points of data. This application system uses Leaflet.js framework, PostgreSQL Database, Apache Tomcat Server and Geoserver. When the map initializes the layer control, address search and well license search are added. The user zooms into a specific spot and turns a layer or multiple layers on. Each layer makes an AJAX (WFS) request to Geoserver. Geoserver pulls the data from PostgreSQL database and sends the information to the Web Application.

  • Date: July 2015
  • Service: Web Application Development
