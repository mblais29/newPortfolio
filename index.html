Parcel Radius Search

Walton's Parcel Radius Search was developed to allow user's the ability to select all parcel's of land within a specified radius. Once the user has completed the radius a javascript function is triggered to capture the points latitude, longitude and distance from the point(radius) in miles. On success, an ajax request to Geoserver is made and returns an array of geojson data. The parcel geometry stored in the PostgreSQL database is rendered in the Leaflet.js framework creating a selection of multiple polygons (parcels of land) within the specified radius.

  • Client: Walton
  • Date: November 2014
  • Service: Web Application Development
  • Code: Preview Code

Routing

Walton's Routing option is an API developed for Leaflet.js. The API was installed into their Web Application and I created the javascript code to capture the user's current latitude/longitude as well as the destination(parcel) coordinates. When the user selects the desired parcel a function is called to retrieved the parcel's center coordinates. Both user's coordinates and the parcel destination coordinates are passed into another function thus generating the quickest route.

  • Client: Walton
  • Date: July 2014
  • Service: Web Application Development
  • Code: Preview Code

Building Location Map

Enbridge building location map was created using Coldfusion to retreive the data from the database, Javascript to run the functions and Leaflet.js framework, HTML and CSS to create the structure and style of the map. Part of the project included displaying important building information. The map used blue pins to display all their building coordinates, users could zoom in and select a pin triggering a pop-up with important information about that building.

  • Client: Enbridge
  • Date: March 2015
  • Service: Web Application Development
